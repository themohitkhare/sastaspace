# Available Prometheus Metrics for Grafana Dashboards
#
# The following metrics are exported by the SastaSpace backend at /metrics
# and can be used to build Grafana dashboards:
#
# Business metrics:
#   - sites_deployed_total           (Gauge)     Number of sites currently deployed
#   - redesign_requests_total        (Counter)   Total redesign requests, labels: status={started,rate_limited,concurrency_limited}
#   - redesign_latency_seconds       (Histogram) End-to-end redesign duration, buckets: 10s,30s,60s,120s,300s,600s
#   - redesign_errors_total          (Counter)   Total redesign errors, labels: phase={crawl,redesign,deploy,unknown}
#   - active_sse_connections         (Gauge)     Number of active SSE streaming connections
#
# Default process metrics (from prometheus_client):
#   - process_cpu_seconds_total
#   - process_resident_memory_bytes
#   - process_open_fds
#
# Suggested dashboard panels:
#   1. Redesign throughput — rate(redesign_requests_total{status="started"}[5m])
#   2. Error rate — rate(redesign_errors_total[5m])
#   3. P50/P95 latency — histogram_quantile(0.95, rate(redesign_latency_seconds_bucket[5m]))
#   4. Active connections — active_sse_connections
#   5. Sites deployed — sites_deployed_total
