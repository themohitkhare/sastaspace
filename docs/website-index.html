sastaspace

Mohit's portfolio + projects as a TUI binary.

Demo

demo recording — try the binary instead

Install

One line — picks the right binary for your platform:

curl -sSf https://sastaspace.com/install.sh | sh

Or via Homebrew:

brew install themohitkhare/sastaspace/sastaspace

Or pick a binary directly from GitHub Releases — macOS (arm64+x86), Linux (x86+arm64), Windows.

Run

sastaspace

Opens the portfolio splash. Use Shift-N notes, Shift-T typewars, Shift-D deck, Shift-A admin, Shift-P portfolio. q to quit.

What's inside

  • portfolio — project cards from a live SpacetimeDB.
  • notes — vim-style two-pane editor with comments.
  • typewars — multiplayer typing game across a contested global warmap.
  • deck — plain-text → ready-to-use audio packs.
  • admin — owner-only btop-style dashboard.
