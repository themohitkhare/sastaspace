**1. Create `backend/Dockerfile`** (DOCK-02):
- Base: `python:3.11-slim` (not alpine â€” Playwright needs glibc)
- Install system deps for Playwright/Chromium: `libglib2.0-0`, `libnss3`, `libnspr4`, `libdbus-1-3`, `libatk1.0-0`, `libatk-bridge2.0-0`, `libcups2`, `libdrm2`, `libxkbcommon0`, `libxcomposite1`, `libxdamage1`, `libxrandr2`, `libgbm1`, `libpango-1.0-0`, `libcairo2`, `libasound2`, `libx11-xcb1`, `libxfixes3`, `fonts-liberation`, `xdg-utils`
- Install `uv` via pip
- Copy `pyproject.toml` and `uv.lock` (if exists), then `uv sync --no-dev`
- Run `uv run playwright install chromium` to install the browser
- Copy `sastaspace/` source directory
- Set `SASTASPACE_SITES_DIR=/data/sites`
- Expose port 8080
- CMD: `uv run uvicorn sastaspace.server:app --host 0.0.0.0 --port 8080`
- Healthcheck: `curl -f http://localhost:8080/ || exit 1` (install curl in image)

**2. Create `web/Dockerfile`** (DOCK-03):
- Use multi-stage build for smaller image:
  - Stage 1 (`builder`): `node:22-alpine`, copy `package.json` + `package-lock.json`, `npm ci`, copy source, `npm run build`
  - Stage 2 (`runner`): `node:22-alpine`, copy standalone output from builder (Next.js standalone mode), copy `public/` and `.next/static`
- For standalone mode to work, add `output: "standalone"` to `next.config.ts` â€” executor MUST update this file
- Set `HOSTNAME=0.0.0.0` env var for Next.js to bind correctly
- Expose port 3000
- CMD: `node server.js` (standalone output)
- Healthcheck: `wget --no-verbose --tries=1 --spider http://localhost:3000/ || exit 1` (wget available in alpine)

**Important for web/Dockerfile**: Next.js standalone requires `output: "standalone"` in next.config.ts. Update `web/next.config.ts` to:
```typescript
import type { NextConfig } from "next";
const nextConfig: NextConfig = {
  output: "standalone",
};

```

**3. Create `docker-compose.yml`** (DOCK-01, DOCK-04, DOCK-05, DOCK-06, DOCK-07):
```yaml
services:
  backend:
    build:
      context: .
      dockerfile: backend/Dockerfile
    ports:
      - "${SERVER_PORT:-8080}:8080"
    env_file: .env
    environment:
      - CLAUDE_CODE_API_URL=http://claude-code-api:8000/v1
      - CORS_ORIGINS=http://localhost:3000
      - SASTASPACE_SITES_DIR=/data/sites
    volumes:
      - sites_data:/data/sites
    networks:
      - sastaspace
    depends_on:
      claude-code-api:
        condition: service_healthy
    healthcheck:
      test: ["CMD", "curl", "-f", "http://localhost:8080/"]
      interval: 10s
      timeout: 5s
      retries: 5
      start_period: 15s
    restart: unless-stopped

  frontend:
    build:
      context: ./web
      dockerfile: Dockerfile
    ports:
      - "3000:3000"
    env_file: .env
    environment:
      - NEXT_PUBLIC_BACKEND_URL=http://localhost:8080
    networks:
      - sastaspace
    depends_on:
      backend:
        condition: service_healthy
    healthcheck:
      test: ["CMD", "wget", "--no-verbose", "--tries=1", "--spider", "http://localhost:3000/"]
      interval: 10s
      timeout: 5s
      retries: 5
      start_period: 30s
    restart: unless-stopped

  claude-code-api:
    build:
      context: .
      dockerfile: claude-code-api/Dockerfile
    ports:
      - "8000:8000"
    env_file: .env
    networks:
      - sastaspace
    healthcheck:
      test: ["CMD", "curl", "-f", "http://localhost:8000/health"]
      interval: 10s
      timeout: 5s
      retries: 5
      start_period: 10s
    restart: unless-stopped

networks:
  sastaspace:
    driver: bridge

volumes:
  sites_data:
```
