# Phase 2 — UI Design Contract

> Visual and interaction contract for the Next.js scaffold phase. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

**Phase scope note:** Phase 2 is a scaffold and wiring phase. It produces a single placeholder page to verify the UI toolchain works. The design system tokens defined here establish the foundation that Phases 3 and 4 will build on.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | shadcn (initialized via `npx shadcn@latest init -d` during this phase) |
| Preset | not applicable (default init, style: new-york, base: zinc) |

**Initialization:** `components.json` will be created during Phase 2 execution. The shadcn gate is deferred to plan execution since `web/` does not exist yet. Expected `components.json` shape: style `new-york`, RSC enabled, TSX enabled, base color `zinc`, CSS variables enabled, aliases using `@/*` import prefix.

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing |
| md | 16px | Default element spacing |
| lg | 24px | Section padding |
| xl | 32px | Layout gaps |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions: 44px minimum touch target for interactive elements (buttons, links) on mobile -- applied via `min-h-11` (Tailwind 44px = 2.75rem).

---

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 16px | 400 (regular) | 1.5 |
| Label | 14px | 400 (regular) | 1.4 |
| Heading | 24px | 700 (bold) | 1.2 |
| Display | 36px | 700 (bold) | 1.1 |

**Font family:** Inter, loaded via `next/font/google` with `subsets: ["latin"]`. Applied to `<body>` in root layout.

**Weight rationale:** Two weights only -- 400 (regular) for running text (Body, Label) and 700 (bold) for emphasis text (Heading, Display). This reduces font file downloads and enforces a clear visual hierarchy through weight contrast alone.

**Phase 2 usage:** The placeholder page uses Display (36px bold) for "SastaSpace" title, Body (16px regular) for subtitle text, and Label (14px regular) for the button.

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `hsl(0 0% 100%)` / `--background` | Page background, main surfaces |
| Secondary (30%) | `hsl(0 0% 96.1%)` / `--muted` | Cards, secondary surfaces, muted text backgrounds |
| Accent (10%) | `hsl(0 0% 9%)` / `--primary` | Primary CTA button, focused input borders |
| Destructive | `hsl(0 84.2% 60.2%)` / `--destructive` | Destructive actions only (not used in Phase 2) |

Accent reserved for: primary CTA button ("Coming Soon" placeholder in Phase 2, "Redesign My Site" in Phase 3), active/focused form input borders, link hover states.

**Note:** These are shadcn/ui zinc theme defaults. The CSS variables are defined in `globals.css` via `:root` and `.dark` selectors, then mapped to Tailwind via `@theme inline`. Dark mode is not in scope for v1 but the token structure supports it.
