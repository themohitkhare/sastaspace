    **Step 1: Update .gitignore FIRST (before npm install creates node_modules)**

    Append these entries to the existing .gitignore (do NOT remove existing Python entries):
    ```
    # Node.js / Next.js
    web/node_modules/
    web/.next/
    web/.next/dev/
    ```

    **Step 2: Scaffold Next.js 16 project**

    Run from repo root:
    ```bash
    npx create-next-app@latest web --typescript --tailwind --eslint --app --src-dir --import-alias "@/*"
    ```

    This creates the `web/` directory with App Router, TypeScript, Tailwind v4, and ESLint. Turbopack is default in Next.js 16 -- no flag needed.

    **Step 3: Initialize shadcn/ui**

    Run from web/ directory:
    ```bash
    cd web && npx shadcn@latest init -d
    ```

    The `-d` flag uses defaults: style `new-york`, base color `zinc`, CSS variables enabled. This auto-detects Tailwind v4 and configures `globals.css` with `@theme inline` and CSS variables for `:root` and `.dark`.

    **Step 4: Install Motion and tw-animate-css**

    Run from web/ directory:
    ```bash
    cd web && npm install motion && npm install -D tw-animate-css
    ```

    **Step 5: Add shadcn Button component**

    Run from web/ directory:
    ```bash
    cd web && npx shadcn@latest add button
    ```

    **Step 6: Verify tw-animate-css is imported in globals.css**

    After shadcn init, check that `web/src/app/globals.css` contains `@import "tw-animate-css"`. If not, add it after the `@import "tailwindcss"` line. The file should have these imports at the top:
    ```css
    @import "tailwindcss";
    @import "tw-animate-css";
    ```

    **Step 7: Create placeholder page (web/src/app/page.tsx)**

    Replace the default page with the placeholder that verifies the toolchain. Must match the UI-SPEC layout contract exactly:

    ```tsx
    import { Button } from "@/components/ui/button";

    export default function Home() {
      return (
        <main className="flex min-h-screen flex-col items-center justify-center bg-background text-foreground">
          <h1 className="text-4xl font-bold mb-4">SastaSpace</h1>
          <p className="text-muted-foreground mb-8">AI Website Redesigner</p>
          <Button size="lg">Coming Soon</Button>
        </main>
      );
    }
    ```

    **Step 8: Update root layout (web/src/app/layout.tsx)**

    Ensure the root layout uses Inter font via `next/font/google` with subsets `["latin"]`, applies it to `<body>` with `antialiased` class. Set metadata:
    - title: "SastaSpace - AI Website Redesigner"
    - description: "See your website redesigned by AI in 60 seconds"

    ```tsx
    import type { Metadata } from "next";
    import { Inter } from "next/font/google";
    import "./globals.css";

    const inter = Inter({ subsets: ["latin"] });

    export const metadata: Metadata = {
      title: "SastaSpace - AI Website Redesigner",
      description: "See your website redesigned by AI in 60 seconds",
    };

    export default function RootLayout({
      children,
    }: {
      children: React.ReactNode;
    }) {
      return (
        <html lang="en">
          <body className={`${inter.className} antialiased`}>
            {children}
          </body>
        </html>
      );
    }
    ```

    **Step 9: Verify next.config.ts is minimal**

    The generated `web/next.config.ts` should be minimal. Ensure it does NOT have any proxy/rewrite rules (anti-pattern: browser talks directly to FastAPI, not through Next.js). Expected content:
    ```typescript
    import type { NextConfig } from "next";
    const nextConfig: NextConfig = {};
    export default nextConfig;
    ```
